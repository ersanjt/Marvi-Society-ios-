## Instagram presence

### [@collabb.me](https://www.instagram.com/collabb.me/)

**Positioning:** Turkish influencer collaboration platform; app-first launch.

**Expected content mix (new brand pattern):**

- App UI screen recordings and mockups
- Creator testimonial Reels (“no more agency emails”)
- Venue partner announcements (cafés, restaurants)
- Launch hype, “Çok Yakında” → live transition
- Turkish-primary captions; emoji-heavy brand voice
- Stories: swipe-up / link to App Store and collabb.me
- UGC reposts from creators after collaborations

**Takeaway for Marvi:** Move fast on Reels showing **instant map flow**; localize for TR; pin App Store link.

### [@thesecretsocietyapp](https://www.instagram.com/thesecretsocietyapp/)

**Positioning:** Luxury lifestyle barter; Dubai-centric exclusivity.

**Expected content mix (mature brand pattern):**

- High-production venue showcases (fine dining, clubs, beach clubs)
- “Do what you can't” lifestyle aesthetic
- Creator POV at VIP experiences
- Event recaps (group influencer nights)
- Aspirational tone; English-primary
- Partner logo grids (venues on platform)
- Social proof: download milestones, event attendance

**Takeaway for Marvi:** Build **aspirational visual language** for venue marketing; repost creator content with collab tags.

## Marvi Society differentiation strategy

1. **Quality over speed:** Don't ship onboarding bugs; TSS took years, Collabb ships fast but gets 1-star onboarding reports.
2. **True cross-platform:** Android from Phase 4 — Collabb is iOS-only today.
3. **Venue web portal early:** Phase 3 — reduce venue friction without requiring mobile.
4. **Transparent creator criteria:** Publish follower + engagement rules like Collabb FAQ.
5. **City depth:** Own Istanbul before UAE expansion (learn from TSS Dubai playbook).
6. **Open architecture:** API contract in repo; professional GitHub ops from day one.
